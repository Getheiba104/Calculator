Calculator App

A responsive, beginner-friendly calculator built using HTML, CSS, and JavaScript.

Live Demo
https://https://getheiba104.github.io/Calculator//calculator/
Technologies Used
- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript (safe functions, no eval)

Features
- Basic arithmetic operations (+ − × ÷)
- Percentage (%) support
- Clear All (AC) button
- Keyboard input support
- Responsive design for all screen sizes
- Clean and modern UI

Keyboard Support
- Numbers: 0–9
- Operators: + − * /
- Decimal: .
- Enter: calculate result
- Backspace: delete last input
- Escape: clear all

Responsive Design
This calculator is fully responsive and works smoothly across all screen sizes.

Highlights:
- CSS Grid-based layout
- Flexible container with max-width
- Touch-friendly buttons
- No horizontal scrolling on mobile
- Tested using Chrome DevTools device toolbar

Supported Devices:
- Mobile phones
- Tablets
- Laptops and desktops
